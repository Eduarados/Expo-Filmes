##  Descrição do aplicativo

Este aplicativo foi desenvolvido em React Native e tem como objetivo exibir informações de filmes utilizando a API do The Movie Database (TMDB).

O sistema permite ao usuário:
- Visualizar filmes populares, mais bem avaliados e lançamentos
- Filtrar filmes por gênero
- Pesquisar filmes em tempo real
- Ocultar conteúdo adulto
- Ver detalhes completos de cada filme

---

##  Funcionalidades

- Listagem de filmes por categorias
- Sistema de busca com filtro em tempo real
- Filtro por gênero (Ação, Terror, Comédia, etc.)
- Filtro de conteúdo adulto (18+)
- Tela de detalhes com:
  - Poster do filme
  - Título
  - Data de lançamento
  - Sinopse
- Navegação entre telas

---

##  Tecnologias utilizadas

- React Native
- React Navigation
- Axios
- API TMDB
- JavaScript
